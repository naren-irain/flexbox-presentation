Flexbox Model

The flexible box model provides an improvement over the block model in that it does not use floats. A flex container expands items to fill available free space, or shrinks them to prevent overflow.

See the Pen Flex box Basic by naren (@narenkrds) on CodePen.

Flexbox Terminology

Once you given an element's display as flex or inline flex,
that element becomes flex container and the direct child elements are flex items.

When to Use Flexbox ?

You can use it to align, order and distribute space among items in a container, when their size is unknown and/or dynamic

{justify-content} // Default: flex-start

This property aligns flex items along the main axis of the current line of the flex container. It helps to distribute the free space left when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size.

Flex Items Properties

order, flex-grow, flex-shrink, flex-basis, flex, align-self

{order} // Default: 0

The Flex-direction property controls the direction in which the flex-items are laid along the main axis.

Flex items order Example:

See the Pen Flex Item - Order by naren (@narenkrds) on CodePen.

{flex-grow} // Default: 0

This property specifies the flex grow factor, which determines how much the flex item will grow relative to the rest of the flex items in the flex container when positive free space is distributed.

Flex grow Example:

See the Pen Flex grow by naren (@narenkrds) on CodePen.

{flex-shrink} // Default: 1

The flex-shrink specifies the flex shrink factor, which determines how much the flex item will shrink relative to the rest of the flex items in the flex container when negative free space is distributed.

Flex shrink Example:

See the Pen Flex shrink by naren (@narenkrds) on CodePen.

{flex-basis} // Default: auto

The flex-basis property specifies the initial size of a flex-item. Before the flex-grow or flex-shrink properties adjust it's size to fit the container or not.

  • When flex-direction is row, flex-basis controls width.
  • But when flex-direction is column, flex-basis controls height.

Flex basis Example:

See the Pen Flex basis by naren (@narenkrds) on CodePen.

{flex} // Default: 0 1 auto

This is the shorthand for flex-grow, flex-shrink and flex-basis combined. The second and third parameters (flex-shrink and flex-basis) are optional.

  • It is recommended that you use this shorthand property rather than set the individual properties.

Flex Example:

{align-self} // Default: auto

This property sets the individual cross axis alignment for the flex items. The value 'auto' inherits the value from the parent element's property align-items.

Useful Tips & Tricks

Vertically center

Before flexbox, vertical centering of dynamic content was a non-trivial task. Now with simple flex properties we can achieve this.

  .vCenter {
    display: flex;
    align-items: center;
    justify-content: center;
  }
  .vCenter.multiple {
    flex-direction: column;
  }

Equal Height

By default all the flex items will be in equal height as the default value for the align-items property is 'stretch'

  .fcBox {
    display: flex;
  }

Full screen view

No more javascript to set the full screen app main content height. Use flex-grow and flex-basis to acheive this.

  .app {
    height: 100%;
    display: flex;
    flex-direction: column;
  }
  .app header {
    flex: 0 1 48px;
  }
  .app footer {
    flex: 0 1 34px;
  }
  .mainContent {
    flex: 1 1 auto;
  }

Transition with flex

Not all the flexbox properties are aupport transition. Transition applicable properties are: flex-grow, flex-shrink, flex-basis, flex

  .fiBox {
    flex-basis: 200px;
    transition: flex-basis 0.5s;
  }
  .parent:hover .fiBox {
    flex-basis: 150px;
  }
  .parent:hover .fiBox:hover {
    flex-basis: 300px;
  }

Margin auto trick

You may noticed there is align-self property to align items in cross axis. But, no justify-self property to align flex items in the main axis.
You can use margin auto for this scenario. 

  .rightChild {
    margin-left: auto;
  }

Margin auto trick

You can center align a single flex child using margin auto.

  .parent {
    display: flex;
  }
  .singleChild {
    margin: auto;
  }

Known issues

  1. Display flex currently not working with button element in firefox and fieldset in most browsers
  2. IE 11 does not vertically align items correctly when min-height is used
  3. In IE10 the default value for flex is 0 0 auto rather than 0 1 auto as defined in the latest spec
  4. Flex-shrink conflicts with percentage units in width/height. For this use flex-basis.
  5. Not all the flexbox properties are aupport transition. Transition applicable properties are: flex-grow, flex-shrink, flex-basis, flex
