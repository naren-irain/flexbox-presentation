Flexbox Model

The flexible box model provides an improvement over the block model in that it does not use floats. A flex container expands items to fill available free space, or shrinks them to prevent overflow.

See the Pen Flex box Basic by naren (@narenkrds) on CodePen.

Flexbox Terminology

Once you given an element's display as flex or inline flex,
that element becomes flex container and the direct child elements are flex items.

When to Use Flexbox ?

You can use it to align, order and distribute space among items in a container, when their size is unknown and/or dynamic

Flex Container Properties

Flex Flow

{justify-content} // Default: flex-start

This property aligns flex items along the main axis of the current line of the flex container. It helps to distribute the free space left when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size.

{align-items} // Default: stretch

This property aligns flex items along the cross axis of the current line of the flex container.

Flex Items Properties

order, flex-grow, flex-shrink, flex-basis, flex, align-self

{order} // Default: 0

{flex-grow} // Default: 0

This property specifies the flex grow factor, which determines how much the flex item will grow relative to the rest of the flex items in the flex container when positive free space is distributed.

Flex grow Example:

See the Pen Flex grow by naren (@narenkrds) on CodePen.

{flex-shrink} // Default: 1

The flex-shrink specifies the flex shrink factor, which determines how much the flex item will shrink relative to the rest of the flex items in the flex container when negative free space is distributed.

Flex shrink Example:

See the Pen Flex shrink by naren (@narenkrds) on CodePen.

{flex-basis} // Default: auto

The flex-basis property specifies the initial size of a flex-item. Before the flex-grow or flex-shrink properties adjust it's size to fit the container or not.

  • When flex-direction is row, flex-basis controls width.
  • But when flex-direction is column, flex-basis controls height.
